The computation of Conway polynomials can be time consuming. Therefore, GAP comes with a list of precomputed polynomials. If a requested polynomial is not stored then GAP prints a warning and computes it by checking all polynomials in the order defined above for the defining conditions. If n ğ‘› n is not a prime this is probably a very long computation. (Some previously known polynomials with prime n ğ‘› n are not stored in GAP because they are quickly recomputed.) Use the function IsCheapConwayPolynomial (59.5-2) to check in advance if ConwayPolynomial will give a result after a short time.
